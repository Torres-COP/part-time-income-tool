# part-time-income-tool
# アルバイト収入 月次レポート生成ツール
◆概要
勤務日をまとめて入力し、深夜手当込みで日別収入と月次レポートを生成するPythonツールです。

◆作成背景
従来の収入官吏では全日入力が必要で入力負担が大きかったため、勤務日のみ入力する方式に改善しました。

◆主な機能
勤務日のまとめ入力
勤務時間（1:45）入力
深夜手当込みの計算
月収の自動計算

◆工夫した点
入力回数の削減、入力形式を時間（分に変更した）、入力エラーを防止

◆今後の改善案
カレンダーでの勤務日の計算
支出管理機能の追加
Webアプリ化
